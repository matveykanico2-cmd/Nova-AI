# 🔥 Real AI Integration для Nova AI

Теперь Nova AI может использовать **настоящий ИИ** для ответов на любые вопросы!

---

## 📋 ДОСТУПНЫЕ ПРОВАЙДЕРЫ

### 1️⃣ Hugging Face (БЕСПЛАТНО) ⭐ РЕКОМЕНДУЕМ

**Плюсы:**
- ✅ Бесплатно
- ✅ Не требуется установка
- ✅ Хорошее качество

**Минусы:**
- ❌ Лимит запросов в день
- ❌ Может быть медленно

**Настройка:**
1. Зайди на https://huggingface.co/settings/tokens
2. Создай аккаунт (если нет)
3. Создай новый токен (Read access)
4. Скопируй токен
5. В Nova AI: Настройки → AI Провайдер → Hugging Face
6. Вставь токен

---

### 2️⃣ Ollama (ЛОКАЛЬНО) 💻

**Плюсы:**
- ✅ Полностью бесплатно
- ✅ Работает без интернета
- ✅ Приватно

**Минусы:**
- ❌ Нужен мощный ПК (8GB+ RAM)
- ❌ Требует установки

**Настройка:**
```bash
# 1. Установи Ollama
# Windows: скачай с https://ollama.com/download
# Mac/Linux: 
curl -fsSL https://ollama.com/install.sh | sh

# 2. Скачай модель
ollama pull llama3.1:8b

# 3. Запусти (автоматически)
ollama serve

# 4. В Nova AI выбери Ollama
```

---

### 3️⃣ OpenAI GPT (ПЛАТНО) 💰

**Плюсы:**
- ✅ Лучшее качество (GPT-4o)
- ✅ Быстро
- ✅ Стабильно

**Минусы:**
- ❌ Платно (~$0.50/1000 запросов)
- ❌ Требуется карта

**Настройка:**
1. Зайди на https://platform.openai.com/api-keys
2. Создай аккаунт
3. Привяжи карту
4. Создай новый API ключ
5. Скопируй ключ (начинается на `sk-`)
6. В Nova AI: Настройки → AI Провайдер → OpenAI
7. Вставь ключ

---

### 4️⃣ Google Gemini (ПЛАТНО) 🔵

**Плюсы:**
- ✅ Хорошее качество
- ✅ Есть бесплатный tier (60 запросов/мин)
- ✅ Мультимодальность

**Минусы:**
- ❌ Требуется аккаунт Google

**Настройка:**
1. Зайди на https://makersuite.google.com/app/apikey
2. Войди через Google
3. Создай API ключ
4. Скопируй ключ
5. В Nova AI: Настройки → AI Провайдер → Gemini
6. Вставь ключ

## ⚙️ НАСТРОЙКА В NOVA AI

1. **Открой настройки** (кнопка ⚙️ в sidebar)
2. **Выбери AI Провайдера** из списка
3. **Введи API ключ** (если требуется)
4. **Нажми "Сохранить"**
5. **Выбери модель "Real AI"** в селекторе моделей
6. **Готово!** Теперь Nova отвечает настоящим ИИ

## 🐛 УСТРАНЕНИЕ ПРОБЛЕМ

### "Не удалось подключиться к Ollama"
```bash
# Проверь что сервис запущен
ollama serve

# Проверь модель
ollama list
# Если нет llama3.1:8b
ollama pull llama3.1:8b
```

### "Hugging Face ошибка"
- Проверь что токен действителен
- Убедись что есть Read access
- Попробуй другую модель

### "OpenAI ошибка 429"
- Превышен лимит запросов
- Подожди или добавь средства на счёт

### "Gemini ошибка"
- Проверь API ключ
- Убедись что API включен в Google Cloud Console

## 📊 МОНИТОРИНГ

Real AI показывает:
- **Provider**: какой провайдер используется
- **Model**: название модели
- **Latency**: время ответа (мс)
- **Tokens**: количество токенов

---

## 🔐 БЕЗОПАСНОСТЬ

- API ключи хранятся **локально** в localStorage
- **Не передаются** третьим лицам
- **Не отправляются** на сервера Nova AI
- Используются **только** для запросов к провайдеру
