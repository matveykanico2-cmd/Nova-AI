# Nova AI - Универсальный Интеллект

Полнофункциональная AI-платформа с поддержкой текста, кода, изображений, голоса и многого другого.

## 🔥 НОВОЕ: Real AI Integration

Теперь Nova AI может использовать **настоящий ИИ** для ответов на любые вопросы!

### Доступные провайдеры:
- **Hugging Face** — бесплатно, хорошее качество
- **Ollama** — локально, бесплатно, приватно
- **OpenAI GPT-4** — лучшее качество (платно)
- **Google Gemini** — мультимодальность (платно)

**Настройка:** Открой ⚙️ Настройки → AI Провайдер → выбери и введи ключ.

📖 **Подробная инструкция:** [REAL_AI_SETUP.md](REAL_AI_SETUP.md)

## 🚀 Быстрый Старт

### Вариант 1: Простой запуск
Просто откройте `index.html` в любом современном браузере (Chrome, Edge, Firefox).

### Вариант 2: Локальный сервер
```bash
# Через Python
python -m http.server 3000

# Через Node.js
npx http-server -p 3000

# Через PHP
php -S localhost:3000
```

Затем откройте http://localhost:3000

## 🐛 Устранение проблем

### Ошибка: "Cannot read properties of null"
**Решение:** Все ошибки исправлены в версии 1.0.1. Обновите файлы и перезагрузите страницу.

### Не работает голосовой ввод
1. Проверьте разрешения браузера для микрофона
2. Используйте HTTPS или localhost
3. Попробуйте Chrome/Edge (лучшая поддержка)

### Не загружаются файлы
1. Проверьте размер файла (< 10MB)
2. Убедитесь что формат поддерживается
3. Попробуйте другой браузер

### Пропадает история
1. Очистите localStorage: `localStorage.clear()`
2. Проверьте что не в режиме инкогнито
3. Обновите страницу (F5)
